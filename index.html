Despre

Rogentos Linux Group este primul grup românesc ce a pornit dezvoltarea sistemelor de operare derivate din Gentoo și pe tehnologiile adiacente și aferente acestora.

Rogentos Linux Group își propune să dezvolte sistemele de operare cu tehnologie Gentoo, numite Kogaion, Argent și ArgOS.

Distribuții

Kogaion

Kogaion reprezintă varianta Desktop, care are ca țintă principală utilizatorul cotidian de Linux.

Argent

Modulele Server sunt vizate de sistemul denumit Argent. Acesta se află încă în dezvoltare, fiind nepublicat și utilizat momentan doar de dezvoltatori. De asemenea, ținta prioritară a sistemului Argent este securitatea națională și cibernetică.

ArgOS

Dispozitivele de tip ARM prezintă o atenție aparte din partea noastră prin tehnologiile oferite de Argos.

Facem următoarele precizări:
x86 reprezintă arhitectura 32 de biți
x64/amd64 reprezintă arhitectura 64 de biți
În dezvoltare folosim 3 medii grafice: Xfce, MATE, LXQt.
